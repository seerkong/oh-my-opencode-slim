# Playwright 浏览器自动化

当任务需要真实浏览器交互时使用此技能。

## 使用场景

- 打开和导航网站
- 填写表单和点击 UI 元素
- 截取屏幕截图
- 验证视觉行为和流程
- 从页面提取数据
- 复现仅在浏览器中出现的 bug

## 注意事项

- 优先使用确定性的、逐步的浏览器操作。
- 保持交互明确且可验证。
- 在关键操作后报告重要的页面状态。
